# Semantic Models

This folder contains the Power BI semantic models for the Laserengrave project.

## Folder Structure

- `definition/` : Internal metadata JSON files.
- `table/` : Individual table definitions (`*.tmdl`) for the model.
- `definition.pbism` : Main semantic model file. Open in Power BI Desktop or Fabric to inspect.
